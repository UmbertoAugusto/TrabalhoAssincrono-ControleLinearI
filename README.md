Esse repositório tem como único objetivo armazenar o código usado no trabalho assíncrono da disciplina de Controle Linear I do curso de Engenharia Eletrônica e de Computação minitrado na Escola Politécnica da Universidade Federal do Rio de Janeiro.
Por completude, o relatório produzido pelo aluno e os gráficos gerados pelo código também estarão presentes.

O código está dividido pelas partes do enunciado, ou seja, em 3 partes. É possível rodar cada parte individualmente, alterando a linha: ```partes = '123'```. Para rodar apenas uma parte, basta deixar apenas o número referente à parte que se deseja executar.

Os resultados numéricos serão impressos no terminal e os gráficos e imagens serão abertos em janelas separadas.
